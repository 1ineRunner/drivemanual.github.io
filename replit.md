# Hutt Valley Drive Manual

## Overview
A static website for a manual driving lesson business serving the Hutt Valley area in New Zealand. The site helps potential customers learn about the service, view pricing packages, and get in touch to book lessons.

## Project Structure
- `index.html` - Main HTML page with all content sections
- `style.css` - Complete styling for the website

## Features
- Responsive design for mobile, tablet, and desktop
- Navigation with smooth scrolling
- Hero section with key value propositions
- "Why Choose Us" benefits section
- "How It Works" 3-step process
- Pricing packages (Starter, Confidence Builder, Full Course)
- Contact form and business information
- Footer with quick links

## Development
This is a static HTML/CSS website served on port 5000 using Python's built-in HTTP server.

## Deployment
Configured for static deployment. The entire root directory is served as the public directory.
